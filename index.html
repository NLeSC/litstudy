filter_duplicates(key=None)¶

Remove duplicate documents from this DocumentSet.

Parameters

key – lambda which takes a document and returns a unique key.

This key is used as an identifier to check if two documents are identical. By default, equivalence is determined based on DOI (if available) or title (if available). :return: A DocumentSet instance

union(other, key=None)¶

Returns the union of this DocumentSet and another DocumentSet and remove duplicate entries. By default, equivalence is checked using the same method as filter_duplicates.

litstudy.plot.plot_affiliation_histogram(docset, top_k=10, ax=None)¶

Plot a histogram of the number of documents published by each affiliation. Note that this is done on a best-effort basis since one affiliation could have multiple spellings (e.g., “University of Amsterdam”, “Universiteit van Amsterdam”, or “UvA”).

Parameters

  • docset – The DocumentSet.

  • ax – The axis on which to plot the histogram, defaults to current axis.

  • top_k – Limit results to the top k institutes.
